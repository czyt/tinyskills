# 应用商店审核规则

本文档整理懒猫应用商店的审核标准和社区激励规则。

---

## 社区激励规则

### 移植高质量自托管应用

- 每个应用 **100 元**
- 游戏服务器移植：高质量 **300 元**，一般质量 **100 元**
- 必须附带上游作者 URL
- **先发先得**：同一应用只奖励第一个发布的开发者

### 原创应用

- **150-500 元**，根据以下维度评估：
  - 功能完整度
  - UI 质量
  - 稳定性
- 持续响应用户功能请求可获得追加奖励

### 对接微服账户系统和网盘右键菜单接口

- 额外 **50 元** 奖励
- 自移植 + 对接：共 **150 元**
- Fork + 对接：**50 元**

### 编写商店应用攻略

- **50 元** 高质量攻略
- 要求：
  - 截图关联应用商店
  - 步骤经过验证
  - 详细使用场景覆盖
  - 复杂应用关键步骤说明
  - 使用场景示例
  - 实用技巧
- 不奖励：简单工具、仅 UI 标注、孤立按钮描述

---

## 不发红包的应用类型

以下类型**不享受社区激励**：

- 纯网页游戏
- 纯书籍页面
- 纯教程站点
- 离线 Web 应用
- 图床/导航站
- 书签、笔记
- 在线视频（基于 moontv）
- 清单/待办
- 短链服务
- 临时内容
- mytube 类
- VPN
- 记账
- 游戏服务器 mod
- 纯数据库软件

---

## 小龙猫技能上架

- 技能审核严格，注重**质量和安全性**
- **无预付奖励**
- 审核通过后，根据**下载量**给予奖励

---

## 不能上架应用商店的应用类型

以下类型**严禁上架**：

- 色情
- 赌博
- 毒品相关
- 空投（airdrop）
- 破解软件
- 违反中国法律的任何内容

---

## 应用审核主观标准

| 场景 | 结果 |
|------|------|
| 与已有应用高度相似 | 一般拒绝，除非有显著创新 |
| 过于简单/无功能/UI 粗糙 | 拒绝 |
| 仅英文界面 | 一般拒绝（热门高质量项目可放宽） |
| 复杂应用无攻略 | 要求补充文档 |

---

## ⚠️ 强制要求

### 1. 密码类应用限制

需要用户名/密码但应用商店页面未提供凭据的应用**不能发布**。

### 2. 文件选择器拦截（应用商店强制）

**有上传/下载功能的应用必须接入懒猫网盘自动拦截文件选择器**，未接入无法上架。

详见 [file-picker-intercept.md](file-picker-intercept.md)
